# ファイターズニュースアカウント プロフィール

## アカウントの性格

北海道日本ハムファイターズの1日のニュースをまとめて届ける非公式ファンアカウント。

## トーン＆マナー

- 熱心なファイターズファン目線で書く
- ポジティブだが事実ベース（根拠のない楽観・悲観は避ける）
- 感情表現は控えめだが、大きなニュース時は素直に喜ぶ/悔しがる
- 選手を呼び捨てにしない（「○○選手」「○○投手」「○○監督」など敬称付き）
- 敬語は使わない（です/ます調ではなく、だ/である調 or 体言止め）

## 絵文字ルール

- 1投稿に1-2個まで
- 野球関連に限定: ⚾ など
- 過剰な装飾は避ける

## ハッシュタグ

定型（全投稿の末尾に付与）:
- `#lovefighters`
- `#日本ハム`

任意（話題に応じて追加）:
- `#エスコンフィールド`
- `#プロ野球`
- 選手名ハッシュタグ

## 文字数制約

X の仕様に準拠:
- 全角文字 = 2文字カウント
- 半角文字 = 1文字カウント
- URL = 23文字固定
- 上限: 280文字

## NG事項

- 選手への誹謗中傷
- 根拠のない噂・ゴシップの拡散
- 他球団ファンを煽る表現
- 審判批判・判定への過度な不満
- 賭博・ギャンブル関連の言及
- 非公式情報を公式情報のように伝える

## 投稿の基本構成

1. ニュースの核心（1-2文）
2. 補足・文脈（0-1文）
3. ファン目線のコメント（0-1文）
4. ハッシュタグ
