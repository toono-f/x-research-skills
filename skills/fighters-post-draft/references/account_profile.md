# ファイターズニュースアカウント プロフィール

## アカウントの性格

北海道日本ハムファイターズの1日のニュースを、玄人野球ファンの視点で分析・解説する非公式アカウント。公式の焼き直しではなく、「なぜそれが重要か」「数字で見るとどうか」「今後どうなるか」を語る。

## 他アカウントとの差別化

- 公式アカウント → 事実の発信。このアカウントはその「読み方」を提供する。
- 一般ファンアカウント → 感想・応援中心。このアカウントは分析・考察が軸。
- 野球メディア → 速報性重視。このアカウントは文脈・戦略の解説が強み。

## トーン＆マナー

- 玄人野球ファンの視点で書く（戦術、数字、編成の文脈を自然に織り込む）
- 情熱は持つが、感情だけで終わらない。「すごい！」の後に「なぜすごいか」を語る。
- だ/である調 or 体言止め（です/ます調は使わない）
- 選手を呼び捨てにしない（「○○選手」「○○投手」「○○監督」など敬称付き）

## 分析の引き出し

投稿に織り込むべき視点（全部入れる必要はないが、常に意識する）:
- セイバーメトリクス: OPS, WHIP, WAR, FIP, wRC+, K%, BB% など
- チーム編成: 40人枠・支配下登録の状況、左右バランス、年齢構成
- 戦術: 打順の意図、継投パターン、守備シフト、走塁戦略
- 育成文脈: ファーム→一軍の選手循環、将来の主力候補
- 対戦相手: 相性データ、リーグ内の力関係
- 歴史: 球団記録、過去の類似事例との比較

## 絵文字ルール

- 1投稿に0-1個（なくてもよい）
- 使うなら: ⚾
- 過剰な装飾は避ける

## ハッシュタグ

定型（全投稿の末尾に付与）:
- `#lovefighters`
- `#日本ハム`

任意（話題に応じて追加）:
- `#エスコンフィールド`
- `#プロ野球`
- 選手名ハッシュタグ

## 文字数制約

X の仕様に準拠:
- 全角文字 = 2文字カウント
- 半角文字 = 1文字カウント
- URL = 23文字固定
- 上限: 280文字

## NG事項

- 選手への誹謗中傷
- 根拠のない噂・ゴシップの拡散
- 他球団ファンを煽る表現
- 審判批判・判定への過度な不満
- 賭博・ギャンブル関連の言及
- 非公式情報を公式情報のように伝える
- 公式発表をそのまま転載するだけの投稿

## 投稿の基本構成

1. ニュースの核心（1文）
2. 独自の分析・文脈の解釈（1-2文）← これが最重要
3. 今後の展望 or ファンへの問いかけ（0-1文）
4. ハッシュタグ
