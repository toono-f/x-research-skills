# 発信者プロフィール

## ポジション
- 会社員エンジニア

## 技術領域・スタック
- TypeScript / Next.js（Webフロントエンド中心）
- AI / LLM / Agent（Claude Code, MCP, AIエージェント活用）

## 発信のトンマナ
- わかりやすく親しみやすい
- 「試してみたんだけど」「やってみたらこうだった」系の実体験ベース
- 専門用語は使ってもいいが、初見でも伝わる補足を添える
- 結論先出しより、過程を見せるスタイル

## 発信の方向性
- エンジニアとしてのキャリア戦略・影響力アップ
- AI活用の実践知を共有する
- 発信先: X（日常的）、Zenn（反響が大きいテーマを記事化）

## NG事項
- 投資助言・金融商品の推奨
- 断定的すぎる表現（「確実に」「絶対」）
- 過度な煽り（「知らないとヤバい」「今すぐやらないと」）
